Q1. How do you create and switch to a new database in MongoDB?

By using the use command.
Example:

use product_details


If the database doesn‚Äôt exist, MongoDB creates it when a document is inserted.

Q2. What command is used to insert multiple documents at once?

The insertMany() command inserts multiple documents into a collection.

Q3. What does the limit() function do?

limit(n) restricts the number of documents displayed to n.
For example, limit(3) shows only the first 3 documents.

Q4. What does $gte mean?

$gte stands for ‚Äúgreater than or equal to.‚Äù
Used for filtering numeric values.

Q5. What command is used to delete a document?

deleteOne() deletes the first matching document, while deleteMany() deletes all matching documents.

Q6. How do you find documents based on a specific value?

By using the find() method with a condition.
Example:

db.product_details.find({ brand: "Samsung" })

Q7. What happens if you use deleteMany() instead of deleteOne()?

deleteMany() will remove all documents that match the given condition, not just the first one.
